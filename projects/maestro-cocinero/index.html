Maestro Cocinero

Portal de recetas personalizadas con buscador avanzado por ingredientes, desarrollado en Next.js y desplegado con Docker Compose sobre PostgreSQL.

Next.jsDockerPostgreSQLTypeScript
Visitar sitio ↗

Características del proyecto

  • Buscador de recetas filtrado por ingredientes disponibles en casa
  • Sistema de personalización: preferencias dietéticas y alergias
  • Generación dinámica de recetas con Next.js App Router y SSR
  • Base de datos relacional PostgreSQL desplegada con Docker Compose
  • Diseño responsive con enfoque mobile-first
